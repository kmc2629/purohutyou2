作品名: Tag Portraits

使用言語: JavaScript / HTML / CSS

概要: 管理人だけが画像を投稿し、閲覧者は各投稿に自分用のハッシュタグや一言メモを付けられるプロフィール型画像アーカイブ。

工夫: 人気タグ・部分一致の類似タグ提案、名前とタグの完全一致 / 部分一致検索、あいうえお順・誕生日順・更新順の並び替えを実装。

改善案: 本番ではFirebaseなどを使い、管理人認証と閲覧者ごとの保存をサーバー側で行う。
